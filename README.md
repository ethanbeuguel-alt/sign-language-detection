# Sign Language Detection (Computer Vision + ML)

## Overview
This project implements a sign language detection system using computer vision and machine learning.

## Features
- Hand landmark detection with MediaPipe
- Gesture classification using a trained Keras model
- Prediction pipeline from images/video

## Tech stack
- Python
- OpenCV
- MediaPipe
- TensorFlow / Keras
